在本地创建远程分支并关联起来

创建本地分支：
git checkout -b dev
此时仅为一个本地分支，还没有同步到远端
git push origin dev
git checkout master
git branch -d dev
git checkout -b dev origin/dev
或者直接
git push --set-upstream origin dev
推送到远端
